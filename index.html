My Skills

For all phases of front end development, I've got you covered.

Ideas

I work with clients to turn concepts into beautiful and functional designs. After all, that's where great user experiences start.

UX/UI

I specialize in designing user interfaces based on intuitive patterns. I aim for minimalistic design schemes that don't make the user think too much.

Code

I am well-versed in HTML, CSS/SASS, and Javascript. I like to write code from scratch, but I can also work with Javascript frameworks like Vue, or CSS frameworks like Bootstrap. I like to keep my code as clean and concise as possible.

Technologies I'm familiar with:

HTML, CSS, SASS, Javascript, jQuery, Vue, Bootstrap, Zurb Foundation, Jekyll, Git, Node/npm, Webpack, and a little React/Redux
